Projeto concebido com a finalidade de adquirir conhecimentos fundamentais em Angular para futuras aplicações. A iniciativa fundamenta-se em tutoriais disponíveis no YouTube, visando implementar efetivamente o uso do framework Angular, inclusive o seu sistema de roteamento para a gestão de rotas no site.

Este Projeto, o HomeWork Login, construido em TypeScript, utilizas as seguintes ferramentas:

  • Angular 17
  • Scss
  • Json Server

Obs: A utilização do Json Server visa a facilidade na comunicação com o Back-End sem precição da construção do mesmo.
